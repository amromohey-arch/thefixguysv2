Google Sheets + Drive Connection

This script saves your daily sales to Google Sheets AND saves receipt images to Google Drive.
  1. Open sheets.google.com in your Fix Guys Gmail → create a new sheet
  2. Click Extensions → Apps Script
  3. Select all → delete → paste the script below
  4. Save (Ctrl+S) → name it anything
  5. Click Deploy → New Deployment
  6. Click ⚙️ → select Web App
  7. Execute as: Me · Access: Anyone
  8. Click Deploy → authorise → copy the Web App URL
  9. Paste URL below and click Save
